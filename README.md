# Diana-Memories

## Adding a large album

For albums with many photos, keep the same structure used by `2023/Memories 2023`:

- Put the full-size images in one album folder.
- Add a matching `thumbs` folder with the same filenames.
- Create an album page with embedded JSON that uses `basePath`, `thumbsPath`, and a filename-only `images` array.
- Add the album link to `index.html` and to the matching year menu on the album pages.
